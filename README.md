# 写给她的四周年纪念网页

世界很大，我们很小

这是我写给我的女主角的四周年纪念网页。

从 `2022.06.25` 到 `2026.06.25`，四年并不只是日历上慢慢翻过去的数字。它是我们从相遇走到相恋，从毕业走向远方，又在一次次奔赴里重新靠近的路。

我想把这些日子认真收起来：那些一起走过的城市，那些隔着屏幕的想念，那些见面前的期待、重逢时的心安，还有我们说过的以后。于是有了这个网页，像一本可以打开的电子纪念册，也像一封写到第四年的长信。

## 在线访问

## 关于这个网页

- 主题：`2022.06.25 → Forever`
- 内容：前言、六章故事、爱情时间轴、四年年轮、照片墙和结语
- 音乐：《有我呢》《稳稳的幸福》

把四年的喜欢、想念、奔赴和未来，笨拙却郑重地写给你。

愿很多年以后再打开它，我们仍然会想起：原来那些平凡又闪光的日子，真的一步一步，把我们带到了今天。
